📍 Overview

This repository contains my solutions and exercises completed during the 1337 Pool (Piscine) — a 4-week intensive coding bootcamp focused on C programming and Unix shell fundamentals. The exercises are organized into daily subjects, each corresponding to specific modules of the 42/1337 curriculum (e.g., C00, C01, C02, etc.).

The goal of this repository is to track progress, reinforce core C concepts, and practice for exams under strict norms (C89/C90 standards, norminette rules, compiler flags). Projects emphasize manual string handling, pointer manipulation, memory logic, and standard input/output handling — without relying on forbidden library functions.

🧠 What Is the 1337 Pool?

The 1337 Pool is an immersive 4-week bootcamp where candidates learn foundational programming skills, primarily in C and Unix shell, through daily exercises and weekly exams. It is a critical step in the admission process for the 1337/42 Network School.

The Piscine structure typically includes:

Daily subjects (“Days”) covering core C topics

Weekly exams to validate understanding

Standardized evaluation through peer review and moulinette checks

Adherence to strict coding norms (Norminette)
